The Declining Loop

The while statement executes a statement or a block of 
statements until a specified expression evaluates to false. 

Example

using System;
class WhileTest 
{
    static void Main() 
    {
        int n = 1;
        while (n < 6) 
        {
            Console.WriteLine("Current value of n is {0}", n);
            n++;
        }
    }
}
