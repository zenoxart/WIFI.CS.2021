Die Abweiseschleife

Mit der while-Anweisung wird eine Anweisung 
oder ein Anweisungsblock ausgeführt, bis 
ein bestimmter Ausdruck den Wert false liefert. 

Beispiel

using System;
class WhileTest 
{
    static void Main() 
    {
        int n = 1;
        while (n < 6) 
        {
            Console.WriteLine("Current value of n is {0}", n);
            n++;
        }
    }
}
